      Theme

      Guide

      Columns — Drag cards between Untouched, Active, Blocked, and Watching. Click + in any column header to add a card. Use ⋮ on columns to rename, add, or remove.

      Stars — Toggle on via card ⋮ menu, then click the star bar to add ★. Click a filled star to remove it.

      Notes vs List Items — Notes are free text, private to the card. List items have a checkbox and priority grid; they appear in the global task list for cross-patient triage.

      Priority dots — 3×3 dot grid on each list item. Click to cycle 0→1→2→…→9→0 dots. Higher = more urgent. Sort the global task list by priority or date.

      Data — All patient data stays in your browser (IndexedDB). Nothing is ever sent to a server. Use Export JSON to back up or transfer between workstations.
